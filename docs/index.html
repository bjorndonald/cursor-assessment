DrawDoughnutChart

This function is for creating a doughnut chart based on data and colors entered. Please make sure the amount of segments match the amount of colors and icons

DrawDoughnutChart
Parameters
element (HTMLDivElement) The DOM element where the chart created
data (Array<{value: number}>) The data which will be used to generate the chart
colors (Array<string>) The colors of each segment
iconSources (Array<string>) The icon which will be placed in each segment.

StackedBarChart
Parameters

{ name: '21-25', value1: 120000, value2: 30000, value3: 30000, value4: 30000 },
]

  <StackedBarChart data={data} />
)
